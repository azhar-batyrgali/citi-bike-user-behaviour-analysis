# citi-bike-user-behaviour-analysis
Excel case study analyzing Citi Bike user behavior, trip patterns and customer segmentation
Project overview

This project analyzes Citi Bike usage patterns to understand customer behavior, trip characteristics and operational demand.

Business questions

Most popular pickup locations

Trip duration differences across age groups

Customer segmentation by user type

Weekday vs weekend usage patterns

Relationship between age and trip duration

Dataset

Public Citi Bike trip dataset.

Methodology

Data cleaning in Excel

Feature engineering (age, age groups, season, weekday)

Pivot table analysis

Visualization of behavioral patterns

Business insight generation

Key insights

Working-age subscribers dominate weekday usage

Casual users show weekend leisure behavior

Highest demand concentrated in several key stations

Age shows limited predictive power for trip duration

Recommendations

Increase bike supply in top-demand stations

Target commuter segment in marketing campaigns
